## 一、宇宙架构

### 1.1 主宇宙（蓝星所在宇宙）
- **物理规则**：遵循经典物理学和量子力学，存在真空零点能、光速限制
- **特殊现象**：特定生命星球（智慧生物众多、科技未达二级）会发生引力紊乱，产生引力空洞，开始流浪
- **暗物质脉络**：宇宙中的特殊结构，流浪星球可从中获取能量，进入高维航行
- **湮灭**：因希格里文明实验（提取真空零点能不归还）导致，以光速扩散，湮灭一切物质
- **结局**：整个可观测宇宙终将毁灭
- **高维航行**：流浪星球通过暗物质脉络进行的超光速旅行，每次航行后速度提升，时间约数月

### 1.2 无垠海（登神时代）
- **本质**：流浪文明的最终归宿，位于未知维度，不在主宇宙正常空间内
- **特征**：海洋平面世界，存在层叠的“投影世界”，法则活跃
- **灵界（太虚废墟）**：后仙古时代九层太虚崩塌后的废墟，神灵可快速穿梭，底层沉淀不灭灵光
- **血海**：不灭血海，由永恒烈阳炼化，九神柱崩塌后仍存在，代表不灭
- **骨海**：不死骨海，与血海类似，代表死亡
- **深渊**：魔神时代的“垃圾场”，填满魔神残念、尸骸、污染，是登神失败者的归宿
- **阴土**：死亡世界，轮回制度的根基，后崩塌为阴土废墟

### 1.3 幻象空间（虚界）
- **创造者**：武神
- **本质**：可以重复启动、演化万千可能的世界，是武神“未来无限”权柄的体现
- **时间限制**：每787年重启一次
- **作用**：从无限的未来中筛选可能性
- **与真界关系**：位于真界之中

### 1.4 真界
- **构成**：雾气空间、黑河、黑山、传法殿
- **特征**：介于虚实之间的特殊空间，存在魔神“秽”的尸骸（黑色山脉）
- **连接**：连接幻象空间和古老月
- **最终**：被李前重塑时间线后失去作用

### 1.5 雾气空间
- **特征**：永恒存在，充满雾气和无尽白骨，黑色石柱林立
- **作用**：各文明留言处，形成“流浪文明网络”
- **云文**：通用文字，自带翻译效果
- **塔尔人文字**：雾气空间通用语言之一
- **黑柱**：记载各文明信息，可被破坏

### 1.6 灵界/太虚
- **本质**：后仙古时代九层太虚崩塌后的废墟
- **特征**：
  - 层叠的海洋世界投影
  - 灵界底层沉淀不灭灵光
  - 灵界主宰推动磨盘碾碎骸骨
- **作用**：神灵可以在其中快速穿梭、战斗

### 1.7 深渊
- **本质**：魔神时代的“垃圾场”——众魔神被屠灭后，它们的残念、尸骸、污染被填入的地方
- **特征**：
  - 充满疯狂、杀戮意志
  - 层叠的深渊世界
  - 存在使徒级怪物（如大乘真仙所化的深渊生物）
- **最终**：被李前的完美一击投入深渊的物质全部消化，深渊消失

### 1.8 阴土
- **本质**：死亡世界，轮回制度的根基
- **历史**：永恒烈阳曾重整阴土，建立轮回秩序
- **最终**：阴土废墟被林宇复苏，用于镇压魔龙

### 1.9 时间线
- **本质**：宇宙历史的记录，可被极致存在观察和干预
- **过去无辩**：过去已定，不可改变
- **未来无限**：未来有无数分支，同时存在
- **魔龙**：可吞噬时间线，破坏历史
- **重塑**：李前以完美一击重塑完整时间线

## 二、力量体系

### 2.1 极致分类（终极存在形态）

#### 过去无辩
> 存在的绝对性。一证永证，其存在已成为宇宙底层规则的常量之一，无法被彻底抹除。不可毁灭、长存不灭、时间免疫，可被镇压但无法消灭。道不绝，故过去可存。
- **定义**：存在的绝对性，一证永证
- **特征**：不可毁灭、长存不灭、时间免疫，可被镇压但无法消灭
- **代表**：魔龙、血海、骨海、古老月
- **本质**：道不绝，故过去可存

#### 当世无敌
>力量的绝对性。在当前时间点上，没有任何存在可以抗衡。力量达到顶峰，可镇压同代一切敌，但只能维持瞬间，之后力量必然跌落。力不永，故当世必衰。
- **定义**：力量的瞬时极致性
- **特征**：当前时间点无可抗衡，可镇压同代一切敌，但只能维持瞬间
- **代表**：永恒烈阳、李前（瞬间）
- **本质**：力不永，故当世必衰

#### 未来无限
> 可能的绝对性。在未来维度拥有无限可能，成长空间没有上限，难以预测。需要被守护，否则无限会变成混乱。变无定，故未来需守。
- **定义**：可能性的无穷性
- **特征**：未来维度拥有无限可能，无法预测，需要被守护
- **代表**：武神、林宇（飞升后）
- **本质**：变无定，故未来需守

#### 极致永恒
> 不灭、不死、不朽中注定的那一丝变数，无限变化中必将坍缩的唯一结果。把握“无辩”中的“变”，捕捉“无限”中的“唯一”，占据当前时刻，是为永恒。道留一线，故永恒可证。
- **定义**：不灭、不死、不朽中注定的那一丝变数，无限变化中必将坍缩的唯一结果
- **特征**：把握“无辩”中的“变”，捕捉“无限”中的“唯一”，占据当前时刻
- **代表**：李前（完美一击瞬间）、林宇（飞升瞬间）
- **本质**：道留一线，故永恒可证

### 2.2 修行体系

#### 武道（古圣时代）
- **境界**：炼体、炼神、化龙武极、负山神台、跃门武圣
- **特征**：依靠天地间魔神气血物质异变，调整自身向魔神靠拢
- **武圣门**：武神与天帝联手打造，镇压魔神意志，后消失
- **武圣**：等同于天尊，玉体金骨

#### 术法诀道（仙古早期）
- **术**：采集自身精气神，表现神奇术式
- **法**：练法，诞生法力
- **诀**：法力外显，掐诀施法
- **道**：得道成尊
- **境界**：炼气、筑基、金丹、元婴、化神、炼虚、返虚、合道、大乘

#### 金丹法（李前改进）
- **特征**：利用“魔性”（先天气）铸就道基，彻底解决魔性隐患
- **境界**：炼气、筑基、金丹、元婴、化神、炼虚、返虚、合道
- **优势**：门槛低，成体系，可批量培养

#### 登神体系（登神时代）
- **核心**：奇术 + 隐秘之弦 + 阶梯
- **阶梯数量**：通常九阶，也有七阶、八阶
- **那扇门**：登神阶梯终点，推开即可成为极致
- **神国**：神灵在世界之外的领域，可演化法则
- **登神幻境**：登阶过程中会进入的心灵考验

### 2.3 奇术
- **雷池相关**：闪电奇术（林宇主修），可克制深渊
- **炼狱相关**：火焰奇术（来自火焰犬），高温焚尽万物
- **风相关**：风灵神羽对应的奇术，可获极速
- **水相关**：智慧之书对应的奇术，增强悟性
- **地相关**：地之杖对应的奇术，掌控大地
- **奇术选择**：决定登神阶梯的性质和数量

## 三、游戏系统

### 3.1 《文明先锋》游戏
- **创造者**：武神（主要）+ 永恒烈阳（共同）+ 月蝉真仙（参与者）
- **终端执行者**：小酒馆酒保
- **根本目的**：解决魔神时代的遗毒，筛选能够承载极致权柄的继承者
- **出现时间**：早于希格里文明实验
- **客观作用**：引导流浪文明前往无垠海，保存文明火种

### 3.2 游戏机制
| 机制 | 作用 |
|------|------|
| **玩家-幸存者绑定** | 玩家操控幸存者，幸存者死亡引发超级地震 |
| **属性面板** | 显示体魄、精神、理智、寿命、自由属性点 |
| **技能栏** | 记录已掌握的技能 |
| **秘钥栏** | 记录已获得的极致权柄（永恒之火、道果、不灭身等） |
| **黑锅献祭** | 投入小黑石（深渊结晶）可获得属性点 |
| **加点** | 玩家可分配自由属性点，提升幸存者属性 |
| **理智值** | 献祭会降低理智，需加点恢复 |

### 3.3 小黑石（深渊结晶）
- **来源**：次级深渊污染 + 红薯种植 + 特殊提炼
- **作用**：投入黑锅可获得属性点
- **代价**：献祭时，周围生物会丢失理智，甚至异变
- **最高产量**：可达数千万颗

### 3.4 永恒之火
- **来源**：永恒烈阳权柄的投影
- **特征**：可传递，拥有者意志坚定、品德高尚
- **作用**：提升精神，延长寿命，可超光速通讯
- **三级火种**：最高等级

### 3.5 火种通讯
- **原理**：通过永恒之火即时传递信息
- **速度**：超光速，无视距离
- **用途**：风筝号与蓝星联系的关键

## 四、关键存在

### 4.1 极致存在

#### 武神
- **极致类型**：未来无限
- **权柄**：演化未来、剑中天地、武圣门
- **布局**：创造幻象空间（虚界），留下古剑，定住无限未来
- **结局**：在无限未来尽头，以模糊背影定住所有未来的支流

#### 永恒烈阳（真仙）
- **极致类型**：当世无敌 + 阴天子
- **权柄**：烈阳、血海、骨海、阴天子
- **武器**：黄泉镇狱杖（可切换为终末掩尘刀）、末日灰烬中剑
- **布局**：炼化血海骨海，重整阴土，拆解权柄融入游戏
- **结局**：堕入深渊，但意志仍在

#### 魔龙
- **极致类型**：过去无辩
- **特征**：肉身不灭，死后产生尸生魂
- **能力**：吞噬历史时间线
- **结局**：被林宇斩首，双翼星璇溃散

#### 古老月（月蝉真仙）
- **极致类型**：过去无辩（魔神）
- **权柄**：隐秘、守护、永寂
- **特征**：屠神时代存活下来的魔神，庇护人类
- **状态**：沉睡消化寂灭、终焉两大魔神
- **结局**：被李前的完美一击击杀，苏想继承权柄

#### 阴天子
- **权柄**：死亡世界之主、轮回
- **特征**：永恒烈阳的另一身份
- **象征**：黄泉镇狱杖

### 4.2 关键人物

#### 李前（4号玩家）
- **身份**：蓝星玩家，最终成为“当世无敌”
- **能力**：永恒之火、道果、超感（精神数千万）、通天祗
- **关键事件**：清泉战役登神失败、与天尊争道、完美三击
- **结局**：身体消散，但永恒之火仍在传承

#### 林宇（4号幸存者）
- **身份**：无垠海幸存者，最终成为“未来无限”
- **能力**：武神神通（演化未来）、不灭血海之主、化物（古剑）、五神器
- **关键事件**：荒岛求生、与塔尔人结盟、登神、飞升
- **结局**：达到极致完美状态后飞升，去向未知

#### 苏想
- **身份**：李前伴侣，月亮选中者
- **能力**：隐秘、守护、永寂三道
- **关键事件**：成为巡狩星新主人、合道、镇压古老月
- **结局**：成为太阴之主，守护新世界

#### 宇航员（月蝉子）
- **身份**：月蝉真仙传人，古老月之使
- **特征**：只剩头颅，神力纯净
- **能力**：相位空间、古老月权柄（可借用）
- **结局**：与古老月深度融合，意识状态未知

#### 天尊
- **身份**：修行之路第一位得道尊者
- **目标**：三道合一，成为新的魔神（取代“秽”）
- **手段**：传法殿培养入道真人，收集众生先天气
- **结局**：争道失败，被李前所杀

#### 汤姆（7号幸存者）
- **身份**：林宇的同伴，7号幸存者
- **特征**：玩家已沉睡，无法加点，最弱小的幸存者
- **能力**：震动感应
- **关键事件**：与林宇汇合、照顾大蜘蛛和大傻鸟

#### 帕维尔（12号幸存者）
- **身份**：北联邦玩家对应的幸存者
- **特征**：在三眼族世界修复战舰，与沙漠人合作
- **能力**：天空之眼（类似超感）

#### 伊格纳（13号幸存者，圣天使）
- **身份**：瓦蓝大陆的幸存者，被信仰扭曲的“天使”
- **特征**：受信仰影响，性格变得神圣、宁静
- **结局**：被塔尔人用仿生机器人取代，恢复正常

### 4.3 塔尔人
- **种族特征**：全身毛发，三根手指
- **文明状态**：星际文明，母星被白色天灾毁灭，仅剩一支舰队
- **关键人物**：红毛学士（顷刻）、黑毛大舰长
- **技术**：可控核聚变、星河主炮、虚拟技术、生物芯片
- **与人类关系**：从敌对到结盟，最终成为盟友

### 4.4 蜥蜴人
- **种族特征**：三眼，厌氧生物，寿命极长（平均2000年）
- **文明特征**：佛系、适应力极强、懒散
- **历史**：曾为星际文明，抵达无垠海后因大公鸡尸体影响退化
- **共享者文明**：蜥蜴人的殖民舰队，后组成共享者联盟

### 4.5 共享者联盟
- **组成**：蜥蜴人 + 另外两个文明
- **目的**：搭乘蓝星前往无垠海
- **结局**：与人类终战协议后，部分成员登陆蓝星进入虚拟社区，主力舰队前往天象-b星系殖民

- **影响**：可摧毁城市，甚至导致文明崩溃

### 11.4 先天气（魔性）
- **本质**：魔神残留的意志碎片，生灵与生俱来
- **作用**：武道时代用于修身修心，修行时代成为阻碍
- **特征**：灵慧越高，先天气越多
- **金丹法**：将先天气铸入道基，化阻碍为资粮

### 11.5 役鬼
- **本质**：特殊灵魂体，可被召唤
- **来源**：真界（雾气空间对岸）
- **召唤方法**：先天气 + 大地阴气 + 永眠道则
- **特征**：刀枪不入，战斗力可观

### 11.6 道孽
- **本质**：魔性未炼化干净，形成的另一个“自己”
- **特征**：偏执、疯狂，与本体争夺控制权
- **解决方法**：彻底炼化魔性

### 11.7 心魔
- **本质**：魔神残余意志的体现
- **特征**：导致修行者疯癫、性情大变
- **来源**：魔神时代的遗毒

### 11.8 不灭灵光
- **本质**：生灵死后遗留，转世轮回的关键
- **位置**：灵界底层，沉淀为“发光沙滩”
- **特征**：理论上不可毁灭
- **例外**：雷池可灭掉不灭灵光（如魔龙尸生魂）

### 11.9 法则（“它们”）
- **本质**：魔神的残骸，是诸道的化身
- **特征**：无形之风，彩色光点，可以交流
- **能力**：可解读名号、传递信息
- **分布**：幻象空间、雾气空间存在，主宇宙没有

### 11.10 潮涌
- **本质**：血海的召唤现象
- **效果**：可召唤散布各处的血海回归
- **使用者**：血海之主

### 11.11 位格
- **定义**：存在在法则中的层级
- **作用**：高位格可压制低位格，可看到更多真实
- **来源**：极致权柄、名号、古老传承

### 11.12 名号
- **定义**：法则对存在的称呼
- **获取**：高位格可感应到
- **特征**：法则会告诉你名号
- **例子**：不灭血海之主、注定的飞升者

### 11.13 隐秘之弦
- **本质**：连接超凡者与万有引力的媒介
- **掌握条件**：精神足够强大
- **作用**：可调动万有引力、感知时空
- **破限后**：可更精细控制

### 11.14 理智值
- **本质**：精神稳定程度的量化
- **上限**：100点
- **降低原因**：献祭小黑石、遭遇超凡干扰、直面高位格存在
- **恢复**：玩家加点、特殊能力（如永恒之火）
- **归零后果**：变成疯子或异化生物

### 11.15 火种等级
| 等级 | 特征 |
|------|------|
| **一级火种** | 初步获得，微弱提升 |
| **二级火种** | 较明显提升，可传递 |
| **三级火种** | 最高等级，可超光速通讯 |

### 11.16 精神属性
- **普通人类**：5-10点
- **超进化者**：30-100点
- **极限超进化者**：300点
- **破限真人**：数千至数万
- **大真人**：10万-40万
- **神灵**：数百万至数千万
- **李前最终**：近4000万

### 11.17 体魄属性
- **普通人类**：5-10点
- **超进化者**：30-80点
- **极限超进化者**：300点
- **破限者**：数千
- **血海生物**：可达数万（如1号达利特近两千）

### 11.18 寿命
- **普通人类**：约100岁（细胞优化后）
- **超进化者**：150-300岁
- **极限超进化者**：500-1000岁
- **破限真人**：1000-3000岁
- **神灵**：数万年起步
- **极致存在**：理论上无限

### 11.19 重力
- **本质**：超越时空与维度的唯一力
- **特征**：可被隐秘之弦调动
- **引力常数G**：理论上可改变，人类尚未实现

## 十六、创作特色与哲学内核

### 16.1 叙事手法
- **双主线并行**：李前（蓝星/幻象空间）与林宇（无垠海）交替推进
- **层层揭秘**：游戏本质、极致分类、武神布局逐步揭示
- **时间跳跃**：常有数十年、数百年的时间跨度
- **多视角**：玩家、幸存者、神灵、凡人视角切换

### 16.2 哲学内核
- **道留一线**：世界不完美，完美才有意义
- **存在的本质**：过去不可改、当世不可久、未来不可定
- **希望的价值**：即使宇宙注定毁灭，仍有一线生机
- **传承的意义**：传薪之火，终燎星海

### 16.3 科学设定
- **硬科幻基础**：真空零点能、量子涨落、相对论效应
- **超凡体系**：在科学基础上叠加超凡规则
- **逻辑自洽**：两种体系相互解释，不矛盾
- **技术推演**：从二级文明到三级文明的完整路径

### 16.4 情感内核
- **李前与苏想**：超越生死的陪伴
- **林宇与宇航员**：从利用到信任到共生
- **章廖东与章程**：父与子的传承
- **宋哲与宋琦**：代际的理解与和解
- **“新人”与“老人”**：两种文明的融合
